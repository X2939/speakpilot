# Submission Checklist

本清单根据七牛云 XEngineer 暑期实训营 FAQ 整理，用于提交前自查。

## 必交材料

- [ ] 公开可访问的 GitHub/Gitee/其他开源平台仓库
- [ ] README 文档
- [ ] Demo 视频链接
- [ ] 报名表中填写所选题目和代码仓库地址

## 仓库要求

- [ ] 仓库在本批次题目发布后创建
- [ ] 提交截止后仓库公开可访问
- [ ] `.env`、API Key、个人敏感信息未提交
- [ ] README 显眼位置包含 Demo 视频链接
- [ ] README 包含运行方式、技术栈、核心模块、后续优化方向

## 开发过程要求

- [ ] commit 时间在本批次开始和截止时间之间
- [ ] 保持持续 commit，不在最后一天一次性导入所有代码
- [ ] 如果使用 PR，PR 描述不为空
- [ ] PR 标题说明新增或修改了什么
- [ ] PR 描述包含功能说明、实现思路、测试方式
- [ ] 主分支始终保持可运行

## Demo 视频要求

- [ ] 视频有声音讲解
- [ ] 展示核心功能和效果
- [ ] 展示场景选择
- [ ] 展示语音或文本输入
- [ ] 展示 AI 对话
- [ ] 展示即时纠错评分
- [ ] 展示课后总结
- [ ] 视频上传到可访问平台，例如 bilibili、网盘等
- [ ] README 中放置 Demo 链接

## 作品有效性

- [ ] 作品方向符合本批次题目：AI 英语口语陪练
- [ ] 不是提交上一批次题目作品
- [ ] 作品可以本地运行
- [ ] 依赖安装和启动命令清晰
- [ ] 说明使用的第三方库、API 和模型
- [ ] 如使用 AI 辅助编程，代码和实现逻辑能自己讲清楚

## 答辩准备

- [ ] 能说明为什么选择英语口语陪练方向
- [ ] 能说明前后端架构和数据流
- [ ] 能说明模型调用和稳定反馈模式
- [ ] 能说明即时纠错评分逻辑
- [ ] 能说明课后总结如何量化能力提升
- [ ] 能说明开发过程中遇到的问题和解决方案
